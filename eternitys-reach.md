🔥 Eternity’s Reach – F2P Quick Guide 🔥

Before Start (2:30 prep):
• Zoom out, check map
• If area is crowded → teleport when battle starts

Early Game – Guards:
• Kill Cesare’s Guards ASAP
• Prioritize Lv2 Guards (more XP)
• Use Search, skip contested guards
• Skill order: Right → Right → Left → Left → Right

Skill Path (IMPORTANT):
1️⃣ Right – +1500 Copper per guard
2️⃣ Right – +25% March Speed
3️⃣ Left – +5000 Copper when occupying a vein (60s CD) ⭐
4️⃣ Left – +20% Gathering Efficiency
5️⃣ Right – +50% Gathering Speed for 60s (MANUAL)

Mid Game – Copper Farming:
• Every 60s: send 1 march to a vein → get 5000 Copper → recall
• Track cooldown using event timer
• Keep killing guards until skills are maxed
• Gather Lv2 veins (Lv1 if no Lv2 available)
• Activate last skill ONLY when all marches are gathering

💥 Fracture Vein Outburst:
• Recall ALL troops before it starts
• Rush Fracture Veins (200 Copper/sec 🔥)
• After event:
– Skills not maxed → kill guards
– Skills maxed → back to Lv2 veins + skills

💡 Tip: You can reach Top 10 without fighting for center (Peak of Eternity).
