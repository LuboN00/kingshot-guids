⚓ Overview

The Buccaneer Bounty runs for 7 days, and the goal is to:
Earn as many Corsair Keys as possible (via Harbor Drill + Daily Tasks)
Use those Keys efficiently (mostly on Splendid chests)
Reach milestone rewards for extra loot
Spend Pearls wisely in the Emporium of Enigma before the event ends

🧭 F2P Optimized Plan
Day 1 — Setup & Stockpile

✅ Complete all daily Harbor Drill tasks (focus on missions granting Corsair Keys).
✅ Don’t spend keys yet — wait until you have at least 30+.
✅ Use your 3 free refreshes to check for Splendid chests. If none appear, stop there.
🎯 Goal: Accumulate keys and observe the refresh pattern.

Day 2 — First Splendid Hunt

✅ Use your 3 free refreshes again to look for Splendid chests.
✅ If you find at least 2 Splendid chests, open them.
✅ If you don’t find any Splendid after free refreshes — save your keys again.
✅ Don’t refresh with paid keys yet; too inefficient early on.
🎯 Goal: Begin earning some Pearls of Enigma slowly.

Day 3 — Milestone Push #1

✅ You should now have 60–80 keys.
✅ Use free refreshes → open Splendid chests first, then Exquisite if milestones are near (e.g., 10 or 20 opens).
✅ Try to reach the 20-chest milestone today.

🎯 Goal: Hit first meaningful milestone rewards and start earning steady Pearls.

Day 4 — Restock Day

✅ Focus on Harbor Drill tasks again.
✅ Save most keys today.
✅ Only open if you find 2+ Splendid chests with free refreshes.
🎯 Goal: Rebuild your key supply and plan pearl spending.

Day 5 — Milestone Push #2

✅ Check chest count — push to 40–50 opens total by end of day.
✅ Don’t use keys on Regular chests unless you’re one or two short of a milestone.
✅ If possible, start buying high-value items from the Emporium (Widgets / Forging Hammers).
🎯 Goal: Secure most milestone rewards and valuable Emporium items.

Day 6 — Optimization

✅ Keep using 3 free refreshes.
✅ Open Splendid / Exquisite chests to maximize Pearl gains.
✅ Spend Pearls on gear / permanent upgrades (avoid decorations unless you have extras).
🎯 Goal: Efficiently use remaining keys without waste.

Day 7 — Final Sweep

✅ Use all remaining keys (no reason to save — event ending).
✅ Spend all Pearls in the Emporium.
✅ Focus on milestone completion and resource conversion.
⚠️ Double-check that no Pearls remain unspent before the event closes.
🎯 Goal: Clean finish — maximize total reward value.

💰 Light Spender Optimized Plan (with Harbor Tome)
Day 1–2 — Double Up

✅ Buy Harbor Tome early to double daily rewards.
✅ Complete all Harbor Drills daily to get maximum Keys.
✅ Use free refreshes to target Splendid chests — open right away.
✅ Push to hit the 10-chest milestone by Day 2.

Day 3–4 — Controlled Spending

✅ You’ll have extra Keys from Tome — use a few refreshes beyond free ones (spend up to 6 Keys max/day for refreshes).
✅ Aim to reach 30–40 chest opens total by end of Day 4.
✅ Start buying key Emporium items: Widgets, Forging Hammers, or Hero Exclusive Gear Fragments.
✅ Avoid cosmetic skins until functional upgrades are done.

Day 5–6 — Milestone Rush

✅ Push toward 70–100 chest opens.
✅ Use purchased Key Packs only if close to a milestone or limited-item goal (like full Hero Gear).
✅ Convert all Pearls strategically:

Priority 1: Widgets

Priority 2: Forging Hammers

Priority 3: Hero Gear Fragments

Priority 4: Skins / cosmetics

Day 7 — Final Optimization

✅ Spend leftover Keys on any remaining Splendid chests.
✅ Empty the Emporium (don’t leave Pearls unused).
✅ Optional: buy 1–2 Key Packs if short of a top-tier Emporium item (e.g., unique gear).
✅ Screenshot rewards summary if tracking your event progress.

🏆 Pro-Level Tricks (Both Types)

Track Refresh Patterns – Splendid chests appear roughly once every 4–6 refreshes. Knowing this helps you decide whether a paid refresh is worth it.

Milestone Timing – Save a batch of keys for Day 3 or Day 5 when milestones reset; you can gain multiple bonus rewards simultaneously.

Emporium Timing – Some servers refresh shop items mid-event (check yours). If so, hold Pearls until the best items appear.

Carryover Keys – Don’t stress if you can’t use all Keys; they roll over into the next Buccaneer Bounty event.

Team Sharing – Compare notes with guildmates — sometimes drop rates for chests vary slightly between refresh cycles; knowing this helps you plan.
